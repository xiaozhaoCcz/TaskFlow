# 🎯 从这里开始

欢迎使用 **Vue 可视化工作流构建器**！

## 🚀 三步快速开始

### 第1步：安装依赖

```bash
npm install
```

### 第2步：启动项目

```bash
npm run dev
```

### 第3步：打开浏览器

自动打开 `http://localhost:3000`，或手动访问该地址。

---

## 🎮 如何使用？

### 1️⃣ 拖拽任务

从 **左侧面板** 拖拽任务到 **右侧画布**

```
左侧任务面板          →  拖拽  →          右侧画布
┌──────────┐                        ┌──────────┐
│ 🌐 打开网页 │         拖拽          │ 🌐 打开网页 │
│ 👆 点击元素 │         ──>          │          │
│ ⌨️  输入文本 │                      │          │
└──────────┘                        └──────────┘
```

### 2️⃣ 配置任务

点击任务卡片，展开配置表单，修改参数

### 3️⃣ 搭建工作流

像搭积木一样，从上到下添加任务

### 4️⃣ 运行测试

点击右上角 **"运行工作流"** 按钮，查看执行效果

## 📦 支持的任务

### 基础任务

- 🌐 **打开网页** - 在浏览器打开指定URL
- 👆 **点击元素** - 点击页面上的元素
- ⌨️ **输入文本** - 在输入框输入文本
- ⏱️ **等待** - 等待指定时间
- 📝 **获取文本** - 提取文本并保存到变量
- 📸 **截图** - 保存页面截图

### 控制流

- ❓ **IF 条件** - 条件判断，支持 THEN/ELSE 分支
- 🔄 **WHILE 循环** - 条件循环，重复执行
- 📋 **FOREACH 遍历** - 遍历数组，批量处理
- 🛡️ **TRY 异常** - 异常捕获，支持 TRY/CATCH

## 💡 简单示例

### 示例1：基础流程

```
1. 打开网页 (https://www.baidu.com)
2. 等待 (2000ms)
3. 输入文本 (#kw: "Vue 3")
4. 点击元素 (#su)
5. 截图 (result.png)
```

**操作步骤：**
1. 拖拽 "打开网页" 到画布
2. 点击展开，设置URL为 `https://www.baidu.com`
3. 依次拖拽其他任务
4. 点击每个任务配置参数
5. 点击 "运行工作流"

### 示例2：条件判断

```
IF (count > 5)
  THEN:
    - 输入文本 (#input: "大于5")
  ELSE:
    - 输入文本 (#input: "小于等于5")
```

**操作步骤：**
1. 拖拽 "IF 条件" 到画布
2. 设置条件：`count > 5`
3. 拖拽任务到 THEN 区域
4. 拖拽任务到 ELSE 区域

### 示例3：循环遍历

```
FOREACH (["Vue", "React", "Angular"] → framework)
  - 输入文本 (#search: {framework})
  - 点击元素 (#submit)
```

**操作步骤：**
1. 拖拽 "FOREACH 遍历" 到画布
2. 设置数组：`["Vue", "React", "Angular"]`
3. 设置变量名：`framework`
4. 拖拽任务到循环体区域

## ❓ 常见问题

**Q: 如何保存工作流？**  
A: 目前工作流保存在浏览器内存中。刷新页面会丢失，建议后续添加导出/导入功能。

**Q: 任务拖不动？**  
A: 确保拖到正确的区域（画布或分支容器），等待区域变蓝色高亮。

**Q: IF条件如何写？**  
A: 使用JavaScript表达式，如：`count > 5`、`text.includes("Vue")`

**Q: 循环执行不了？**  
A: 检查循环条件和最大迭代次数设置。

**Q: 如何使用变量？**  
A: 使用"获取文本"任务保存变量，在后续任务或条件中使用变量名引用。
